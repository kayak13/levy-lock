# LevyLock™ Landing Page

A premium, mobile-first React landing page for LevyLock™ judgment enforcement services.

## Features

- **Single-page landing site** with 10 comprehensive sections
- **Interactive fee calculator** with live updates
- **Comprehensive intake form** with file upload capabilities
- **Mobile-first responsive design** with luxury styling
- **SEO optimized** with proper meta tags and JSON-LD structured data
- **Framer Motion animations** for subtle, tasteful interactions
- **shadcn/ui components** with custom gold accent theme
- **Form validation** using react-hook-form + zod
- **Web3Forms integration** for form submissions

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation & Running

```bash
# Install dependencies
npm install

# Start development server
npm run dev
